A generator of the letter-receiving address for NCTU Students

Input Format
The input is the information of a student, containing name, dormitory number, and student ID.

Output Format
The format is 1001 University Road, Hsinchu, Taiwan 300, ROC, dorm < dormitory_number >, (< student_ID >) < first_name > < last_name >

Sample Input 0
Charles Pearson 7 0123456

Sample Output 0
1001 University Road, Hsinchu, Taiwan 300, ROC, dorm 7, (0123456) Charles Pearson
