A string processor that takes a line as input and outputs the input string by removing whitespace(maybe more than one or no whitespace) that appear at the beginning or end of the string, reverting the string.

Input Format
A sentence with no or mulitple whitespace that appear at the beginning or end.

Output Format
Remove the whitespace that appear at the beginning or end of the sentence, then reverse the string.

Sample Input 0
 .enituor yliad ruoy ni neddih si erutuf ruoy fo terces ehT 

Sample Output 0
The secret of your future is hidden in your daily routine.
