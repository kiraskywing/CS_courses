The first integer of input is the number of queries. A query can be of six types:

a x (Adds an element x at the end )
e x (Erase element x)
r (Reverses the vector)
d (Delete the last element)
s (print the vector size)
p (print the vector)

Input Format
6 a 2 a 11 a 7 d s p

Output Format
2 2 11

Sample Input 0
6
a 2 a 11 a 7 d s p 

Sample Output 0
2 2 11
